发布时在 Git Bash 中运行：

```bash
bash scripts/release.sh
```

脚本会生成 changelog、提交文档、创建 tag，并在确认后推送分支与 tag。tag 推送后由
`.github/workflows/release-vscode.yml` 构建 VSCodemo 并发布产物。
